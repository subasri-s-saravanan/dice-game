# 🎲 Dice Game

A simple web-based dice game built using HTML, CSS, and JavaScript.

## 🕹️ How to Play

- Open the web page.
- Click **Refresh** or press **F5** to roll the dice.
- Two dice will appear with random values.
- The player with the higher number wins!
